Sobre os primeiros compuradores

Os primeiros computadores eram maquinas antigas e pesadas, na epoca em questão era a tecnologia do momento, eles tinham como objetivo calculos rapidos e precisos, tudo isso se deu por conta da guerra, com o objetivo de fazer calculos que pudessem levar as tropas a vitoria. Dentre os computadores criados havia o mark I, Eniac entre outros. Esses computadores eram gigantes, ocupavam salas inteiras, além disso trabalhavam com o sistema de valvulas, o que acaba esquetando muito e com isso atraia muitos insetos, gerando assim o termo BUG mais conhecido como inseto.

Como surgiu a internet

A internet surgiu como uma solução para a comunicação de longa discia e transmissão de dados , devida a guerra fria que não era uma guerra de armamentos e sim de tecnologia, na epoca os estados unidos queriam está a frente da antiga união sovietica, e com isso foi prosposto uma forma de fazer a comunicação entre computadores para a troca de informações, na epoca em questão foi criada a DARP. A primeira tentativa de comunição foi feita em 1969 entre a universidade da California e o Instituto de pesquisa Stanford. Em meio a isso foi criado o protocolo TCPI/IP que usado até os dias de hoje. Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW) Primeiro navegador chamado WorldWideWeb.

Conceito de serves

Os servidores são maquinas utilizadas para o armazenamento de informações, quando se tem um servidor e possiveis armazenar os dados para poder se acessar atráves de qualquer lugar com internet, dentro os servidores existentes existem alguns com funcionalidades diferentes. Ex, servidor de e-mail, arquivos, streaming e web. Além disso, os servidores trabalham com sistema desenvolvidos para eles proprios, sendo eles servidores linux e windows. Ou seja, dentro do servidor a um sistema operacional e um serviço de servidor web. Os dois trabalham juntos para transmitir as informações que o dispositivo está que o acessou está requisitando. Os servidores podem ser tanto local ou servidores remotos, onde e possivel alugar contratar pagando uma assinatura mensal.

Tags

h1 - Tag responsavel pelo titulo do site, sendo a maior fonte entre todas as tags

h2 - tag responsavel pelo subtitulo do site, sendo a segunda maior fonte entre todas as tags

Além disso, existem outras tags da mesma familia do h1, indo até h6

u - tag responsavel por deixar as letras em negrito. Porem, atualmente há a tag strong que tem a mesma funcão

i- tag responsavel por deixar as letras em Italico

ol - tag que serve para criar um lista ordenada. Ou seja, uma lista numerada, mais para isso e preciso que seja feita com o auxilio de outra tag, sendo ela a tag l

li - tag usada para a criação de lista

ul - tag usada para a criação de lista não ordenadas

p - tag utilizada para criar paragrafos

a - tag utilizada para a criação de ancoras e links

sup - tag utilizada para elevar nomes ou letras

sub - tag utilizada para deixar letras mais embaixo
